# CeducaQuiz

CeducaQuiz é uma plataforma de gestão educacional voltada para a automação e simplificação de avaliações escolares. O projeto visa otimizar o fluxo de trabalho docente, permitindo a criação e aplicação de provas interativas com foco em engajamento e integração entre turmas.

O fator diferencial do nosso projeto é os seguintes pontos: 

-> Sistema de Gamificação: Integração de um ranking competitivo para aumentar o engajamento dos alunos durante as atividades.

-> Gestão de Mídia: Suporte para inclusão de imagens nas questões, permitindo avaliações mais dinâmicas e visuais.

-> Recuperação de Acesso: Implementação de sistema de e-mail automatizado para recuperação segura de senhas, através de uma biblioteca presente no php chamado PHPMailer.

-> Interatividade em Tempo Real: Cronometragem através de motores JavaScript.

O CeducaQuiz foi construído utilizando tecnologias modernas de desenvolvimento web, focando em performance e experiência do usuário:

-> Backend: PHP – Responsável por toda a lógica do projeto, processamento de dados e comunicação com o servidor.
-> Banco de Dados: MySQL – Utilizado para a modelagem e persistência de dados de alunos, turmas e avaliações.
-> Armazenamento de dados não relacional: JSON - Usado para armazenar valores dentro do sistema de quiz, antes de ser enviado para o banco de dados, mas também armazenar dados das perguntas e respostas

-> Frontend:

HTML5 & CSS3: Estruturação semântica e estilização responsiva.
JavaScript: Implementação de camadas de interatividade, incluindo sistemas de cronometragem em tempo real, validações dinâmicas e o motor do quiz.
Design & UX: Figma – Utilizado para a criação de protótipos de alta fidelidade antes do desenvolvimento, garantindo uma interface intuitiva.
